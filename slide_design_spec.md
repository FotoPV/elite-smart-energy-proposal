# Elite Smart Energy Solutions Slide Design Specification

## Extracted from Best Example: In-Depth Bill Analysis & Solar Battery Proposal - Paul Stokes (SA)

### Color Palette (EXACT - LOCKED IN)
- **Background**: Pure Black (#000000) - DARKEST BLACK
- **Primary Accent (Aqua)**: #00EAD3 - Used SPARINGLY for logo, graph highlights, key savings values, checkmarks
- **Secondary Accent (Orange)**: #f36710 - Headings, key numbers, chart bars, warning icons, negative values
- **Text (White)**: #FFFFFF - Body text, descriptions, large hero numbers
- **Text (Ash/Gray)**: #808285 - Labels, secondary text, table headers

### Typography (LOCKED IN)
- **Main Headings**: NextSphere-ExtraBold (ALL CAPS, large, italic/slanted style, white)
- **Section Labels**: Urbanist-SemiBold (ALL CAPS, letter-spacing: 0.2em, aqua for subtitles)
- **Body Text**: GeneralSans-Regular (white)
- **Key Metrics/Numbers**: NextSphere-ExtraBold (large, white or aqua for savings)

### Slide Structure (16 Slides - SA Example)

1. **Cover Slide**
   - Customer name large

   - Logo top-right

3. **Usage Analysis**
   - Bar chart with monthly kWh consumption
   - Peak/average stats
   - Orange bars, aqua for highlight month

4. **Current Situation Summary**
   - Two-column layout
   - Key metrics on left
   - Challenges/opportunities on right

5. **Strategic Assessment - Battery Storage Investment**
   - Two columns: KEY ADVANTAGES (aqua checkmark) | CONSIDERATIONS (orange warning)

   - Product image on left (Sigenergy SigenStor)
   - Specs on right: Total Capacity (large aqua number), Inverter, Modules, Technology
   - "Why this capacity?" stacked bar (Home/EV/VPP)

7. **Proposed Solar PV System**
   - System size (large number), Panel count, Panel power
   - "Why AIKO?" explanation box
   - Performance & Warranty section with aqua checkmarks

8. **VPP Provider Comparison**
   - Table: Provider | VPP Model | Gas Bundle | Est. Annual Value | Strategic Fit
   - Recommended row highlighted with aqua border
   - Strategic fit badges: EXCELLENT (aqua), GOOD (green), MODERATE (orange), COMPLEX (gray)

9. **VPP Recommendation**
   - Selected partner large (e.g., "ENGIE")

10. **EV Analysis - Low KM Vehicle**
    - Vehicle image on left

11. **Total Savings Summary**
    - Stacked bar chart (Aqua: Solar & Battery, White: EV, Orange: VPP)

12. **Financial Summary & Payback**
    - Two columns: Investment Breakdown | Projected Returns

13. **Recommended Roadmap**
    - Timeline with 4 steps: Approval & Finance → Installation → VPP Activation → EV Integration
    - Aqua/orange progress line
    - Each step has icon, title, description, timeline

14. **Conclusion**
    - Three feature cards: Maximize Returns (aqua border), Secure Power (white border), Future Ready (orange border)
    - Large quote in center

15. **Contact/Thank You** (if present)

16. **Appendix/Additional Data** (if needed)

### Key Design Rules (MANDATORY)
1. Aqua (#00EAD3) used ONLY for: logo, totals, savings values, positive highlights, checkmarks
2. Orange (#f36710) used for: headings, key numbers, chart bars, warnings, negative values
3. White for body text, large hero numbers, descriptions
4. Gray (#808285) for secondary labels, table headers
5. Tables have thin white/gray borders, no heavy styling
6. All caps for section labels with wide letter-spacing (0.2em)
7. Large contrast between hero metrics and supporting text
8. Stacked bar charts use: Aqua (primary), White (secondary), Orange (tertiary)
9. Two-column layout: left sidebar with key metric, right with details/table
10. Cards have thin aqua/orange/white borders based on category
11. NO BLUE colors anywhere - strictly forbidden
12. Rounded corners on buttons, badges, and cards
13. Copyright: "Elite Smart Energy Solutions - Architect [Consultant Name]" at bottom

### Layout Patterns

#### Hero Metric Pattern
```
[LABEL - small, gray, uppercase]
[LARGE NUMBER - white or aqua, NextSphere font]
[UNIT - smaller, white]
```

#### Feature Card Pattern
```
┌─────────────────────────┐
│  [ICON - aqua/orange]   │
│  TITLE - white, caps    │
│  Description - gray     │
└─────────────────────────┘
Border: 1px solid aqua/orange/white
```

#### Table Pattern
```
HEADER ROW - gray text, uppercase
─────────────────────────────────
Data Row 1 - white text
Data Row 2 - white text
─────────────────────────────────
TOTAL ROW - aqua text (if positive)
```

#### Strategic Fit Badges
- EXCELLENT: Aqua background, dark text
- GOOD: Green background, dark text
- MODERATE: Orange background, dark text
- COMPLEX: Gray background, light text
