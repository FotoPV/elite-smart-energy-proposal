## KEY DESIGN PATTERNS (confirmed from Paul Stokes):
1. Headings: MASSIVE white NextSphere-ExtraBold, ALL CAPS
2. Subtitles: AQUA, right-aligned, Urbanist style (sometimes italic)
3. Table headers: AQUA uppercase small text
4. Dollar amounts: WHITE bold for regular, AQUA for savings/totals, ORANGE for payback/alerts
5. Insight cards: Dark grey bg, orange or aqua left border, "KEY INSIGHT"/"EFFICIENCY INSIGHT"/"KEY FINDING" in AQUA NextSphere
6. Highlighted values in body text: AQUA or ORANGE bold
7. Charts: Aqua for customer/proposed, grey/dark for comparison, orange for current/costs
8. Thin aqua horizontal line separators under headings
9. Background: Pure black throughout
10. Very generous spacing, clean minimal layout

## KEY DESIGN RULES CONFIRMED:
- NextSphere-ExtraBold: ALL headings, ALL CAPS, massive size
- GeneralSans-Regular: Body text, descriptions, table data
- Urbanist-SemiBold: Subtitles (right-aligned, AQUA), labels, badges
- Colors: BLACK bg, AQUA (#00EAD3) for savings/positive/subtitles, ORANGE (#E8731A) for costs/alerts/accents, WHITE for data values, GREY (#808285) for labels/secondary
- Insight cards: Dark grey bg (#1a1a1a), colored left border (aqua or orange), heading in AQUA NextSphere
- Tables: Grey uppercase headers, thin row separators, aqua/orange/white values
- Charts: Aqua for proposed/savings, orange for costs/VPP, white/grey for comparison
- Thin aqua horizontal line under every heading
